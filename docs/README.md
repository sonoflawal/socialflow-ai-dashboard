## 💡 Key Concepts

### Wallets
Non-custodial Stellar wallets (Freighter, Albedo) that you control. SocialFlow never has access to your private keys.

### Testnet vs Mainnet
- **Testnet**: Free testing environment with no real value
- **Mainnet**: Production network with real XLM

### XLM (Lumens)
Native cryptocurrency of Stellar network. Required for:
- Transaction fees (~0.0001 XLM)
- Account reserves (1 XLM minimum)
- Trustlines (0.5 XLM each)

### Trustlines
Permission to hold specific assets. Required before receiving custom tokens or USDC.

### Smart Contracts
Self-executing code on Soroban (Stellar's smart contract platform) that automates campaign rewards.

### IPFS
Decentralized storage for NFT content. Files are permanent and content-addressed.

## 🛡️ Security Best Practices

### Do's ✅
- Backup your secret key securely
- Verify all transaction details before signing
- Use Testnet for learning and testing
- Enable 2FA on social media accounts
- Keep software and extensions updated

### Don'ts ❌
- Never share your private/secret key
- Don't approve suspicious transactions
- Avoid using public computers for wallet access
- Don't store large amounts on Testnet
- Never screenshot your secret key
